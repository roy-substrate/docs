# Authentication

The Substrate API uses API keys for authentication. All API requests must include your API key in the request headers.

## Getting Your API Key

1. Sign up for a Substrate account at [https://app.getsubstrate.co](https://app.getsubstrate.co)
2. Navigate to the API Keys section in your dashboard
3. Generate a new API key for your application
4. Store your API key securely

<Warning>
  Keep your API key secure and never expose it in client-side code. API keys should only be used in server-side applications.
</Warning>

## Authentication Methods

### Bearer Token (Recommended)

Include your API key in the `Authorization` header using the Bearer token format:

```bash
curl -X POST https://api.substrate.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4",
    "messages": [
      {
        "role": "user",
        "content": "Hello, world!"
      }
    ]
  }'
```

### API Key Header

Alternatively, you can include your API key using the `X-API-Key` header:

```bash
curl -X POST https://api.substrate.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "model": "gpt-4",
    "messages": [
      {
        "role": "user",
        "content": "Hello, world!"
      }
    ]
  }'
```

## Code Examples

<CodeGroup>

```python Python
import requests

api_key = "your-api-key-here"
base_url = "https://api.substrate.ai/v1"

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}"
}

response = requests.post(
    f"{base_url}/chat/completions",
    headers=headers,
    json={
        "model": "gpt-4",
        "messages": [
            {"role": "user", "content": "Hello, world!"}
        ]
    }
)

print(response.json())
```

```javascript Node.js
const axios = require('axios');

const apiKey = 'your-api-key-here';
const baseURL = 'https://api.substrate.ai/v1';

const headers = {
  'Content-Type': 'application/json',
  'Authorization': `Bearer ${apiKey}`
};

const response = await axios.post(
  `${baseURL}/chat/completions`,
  {
    model: 'gpt-4',
    messages: [
      { role: 'user', content: 'Hello, world!' }
    ]
  },
  { headers }
);

console.log(response.data);
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    apiKey := "your-api-key-here"
    baseURL := "https://api.substrate.ai/v1"
    
    payload := map[string]interface{}{
        "model": "gpt-4",
        "messages": []map[string]string{
            {"role": "user", "content": "Hello, world!"},
        },
    }
    
    jsonPayload, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", baseURL+"/chat/completions", bytes.NewBuffer(jsonPayload))
    
    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("Authorization", "Bearer "+apiKey)
    
    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer resp.Body.Close()
    
    // Handle response
}
```

</CodeGroup>

## Error Responses

If authentication fails, you'll receive a `401 Unauthorized` response:

```json
{
  "error": {
    "message": "Invalid API key provided",
    "type": "authentication_error",
    "code": "invalid_api_key"
  }
}
```
