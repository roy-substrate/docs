# Substrate API Overview

Welcome to Substrate, a comprehensive conversational AI platform designed to power intelligent applications with advanced language model capabilities.

## What is Substrate?

Substrate is a modern API platform that provides:

- **Conversational AI**: Advanced chat completion endpoints
- **Context Management**: Intelligent conversation memory and context handling
- **Multi-Model Support**: Access to various language models
- **Real-time Analytics**: Comprehensive usage tracking and insights
- **Scalable Architecture**: Enterprise-ready infrastructure

## Getting Started

To start using the Substrate API:

1. **Sign Up**: Create your account at [https://app.getsubstrate.co](https://app.getsubstrate.co)
2. **Authentication**: Set up your API credentials
3. **First Request**: Make your first chat completion call
4. **Explore Features**: Try conversations, context management, and analytics

<Note>
  All API endpoints require authentication using your API key. See the [Authentication](/substrate/authentication) section for details.
</Note>
