# Usage & Analytics API

The Usage & Analytics API provides insights into your application's usage patterns and performance metrics.

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `start_date` | string | Yes | Start date for analytics data (ISO 8601 format) |
| `end_date` | string | Yes | End date for analytics data (ISO 8601 format) |
| `metric` | string | No | Specific metric to retrieve (defaults to all metrics) |
| `granularity` | string | No | Data granularity: `hour`, `day`, `week`, `month` (default: `day`) |
| `filter` | string | No | Filter criteria for the data |
| `limit` | integer | No | Maximum number of records to return (default: 100, max: 1000) |

## Example Request

```http
GET /api/v1/usage-analytics?start_date=2023-01-01&end_date=2023-01-31&granularity=day&limit=100
Authorization: Bearer your-api-token
Content-Type: application/json
```
